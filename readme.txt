Данное приложение предназначено для размещения личного файлового хранилища, реализованного в виде веб приложения. Пользователю предоставляется 
возможность создавать древовидную структуру каталогов, загружать и хранить файлы. 

1. В директории back-file-server/back_file_server создаём файл .env, в котором прописываем ваши параметры "SECRET_KEY, DEBUG, ALLOWED_HOSTS". 
Данные параметры будут применяться в настройках проекта settings.py.

2. После клонирования репозитория и настройки виртуального окружения необходимо выполнить миграции в базу данных:
- py manage.py makemigrations
- py manage.py migrate
Также для доступа к admin панели необходимо создать суперпользователя: py manage.py createsuperuser
Для запуска сервера выполняем команду: py manage.py runserver


3. Для запуска фронтенда переходим в каталог front-file-server (где расположен файл package.json). Выполняем команду npm run serve
